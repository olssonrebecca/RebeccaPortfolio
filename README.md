# Portfolio

My portfolio is designed and developed by me, Rebecca Olsson.
Vue application. 


## Project Setup

```sh
npm install
npm i tsparticles 
npm i vue3-particles
npm i tsparticles-slim
npm install -D sass-embedded
npm i @iconify/vue
npm i axios
npm add radix-vue
```
### Compile and Hot-Reload for Development

```sh
npm run dev
```
